Player Statistics

The Player Statistics tab allows you to select a player in the Hand History Database and then look at detailed information for that player. The information in this tab is separated into the following components:

  • Players
  • Sessions
  • Filter Games
  • Totals
  • Bankroll Graph and Graph Options
  • Actions

Players

The Players list allows you to select a specific player in the Hand History Database. Selecting a player in the players list, displays detailed information about that player on the Players tab.

Any player that has ever participated in a hand recorded in the Hand History Database appears in the players list. This includes your own profiles, opponents you have faced while playing Poker Academy Online, and computer opponents you have played in Poker Academy ring games and tournaments.

Tip: To make it easier for you to find your own profiles (i.e. heroes), you can check the Show Heroes Only checkbox below the players list which causes the players list to be filtered to only display hero profiles.

Sessions

The Sessions list displays a list of sessions that the selected player in the player list has played. If no filters are used, then all sessions played by the selected player are shown. Otherwise, only sessions matching the filter criteria are shown.

Each session entry in the session list provides some summary information about the session:

  • the starting Date of the session
  • the Site where the session was played - this will be either PA - Poker Academy 2.0 for ring games and tournaments played on your own computer, or PA Online - Poker Academy Online for ring games and tournaments played on the free Poker Academy internet servers
  • the name of the Table played at during the session
  • the size of the Stakes during the session
  • the number of Hands in the session where the selected player was dealt cards
  • the Net $ Won by the selected player during the session

Tip: The columns in the session list can be sorted by clicking on the column title.

Tip: As a short-cut, double-clicking a session's row will bring up a Hand History Browser for all hands in that session.

In the sessions panel, there are toolbar buttons that perform specific operations on the selected session entries. If no session entry is selected, the buttons will apply to all rows in the table. The buttons and their operations are as follows:
The Hand History button will bring up a Hand History Browser showing only the hands in the selected sessions.
Export transcripts of the hands in the selected sessions to a text file.
Permanently deletes all of the hands in the selected sessions from the database.

Filter Games

The Filter Games panel allows you to filter which sessions appear in the session list. For each checkbox, if it is selected, then sessions of that type are allowed to be displayed in the session list. If it is not selected, then sessions of that type are not displayed in the session list. The four checkboxes are:

  • Ring - filters ring game sessions
  • Tourney - filters tournament sessions
  • Limit - filters fixed limit sessions
  • No Limit - filters no limit sessions

Tip: If you deselect both Ring and Tourney, or both Limit and No Limit at the same time, your entire session list will be filtered to be empty.

Totals

The Totals panel provides summary information for your selected sessions including the total number of hands played by the selected player in the sessions, and the amount of money that player won in the selected sessions.
